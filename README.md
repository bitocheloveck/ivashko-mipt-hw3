# Books Scraper
Учебный проект для автоматического сбора данных о книгах с сайта Books to Scrape.

## Описание
Этот проект реализует систему для:
- Парсинга данных о книгах (название, цена, рейтинг, описание, наличие)
- Автоматического сбора данных со всех страниц каталога
- Ежедневного автоматического обновления данных
- Тестирования функциональности парсера

## Структура проекта

## Установка
### Клонируйте репозиторий:
git clone https://github.com/bitocheloveck/ivashko-mipt-hw3.git
cd ivashko-mipt-hw3

### Установите зависимости:
pip install -r requirements.txt

## Использование
### Запуск парсинга:
scraper.py

### Запуск тестов:
tests

### Автоматический режим:
Скрипт настроен на автоматический запуск каждый день в 19:00. 
Для остановки нажмите Ctrl+C.

## Функциональность
* get_book_data(url) - извлекает данные об одной книге
* scrape_books(is_save=True) - собирает данные со всех страниц каталога
  Автоматическое сохранение результатов в artifacts/books_data.txt
* Система повторных попыток при сетевых ошибках

## Результаты
Данные сохраняются в формате JSON и содержат:
- Название, цену, рейтинг книги
- Описание и количество в наличии
- Дополнительную информацию (UPC, тип продукта и др.)

## Дополнительные комментариии
1. Содержание кода в HW_03_python_ds_2025.ipynb и scraper.py имеет несколько отличий:
    - время сохранения (в ноутбуке время стоит для проверки, в scraper.py - 19:00);
    - директория сохранения (в ноутбуке в ту же директорию, в scraper.py - в папку artifacts);
    - возможны некоторые иные различия, не влияющие на работу кода.
2. Данные с сайта преобразуются в числовые значения, где это ожидается (рейтинг, наличие, стоимость и налоги). Также отдельно выведена валюта стоимости.
3. Значения преобразуются в единый словарь (без подсловаря, как это сделано на сайте), в файл сохраняются в формате json для простоты чтения.
